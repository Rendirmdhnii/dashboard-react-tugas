## 🛠️ Cara Menjalankan Project di Laptop Kalian (Wajib Baca!)

Buat anak-anak yang mau lanjutin kodingan atau sekadar ngetes jalanin web-nya di laptop masing-masing, ikuti urutan langkah di bawah ini (jangan ada yang kelewat!):

### Langkah 1: Clone Repositori
Buka terminal/CMD di laptop kalian, lalu ketik perintah ini untuk mengunduh kode dari GitHub:
```bash
git clone [https://github.com/Rendirmdhnii/dashboard-react-tugas.git](https://github.com/Rendirmdhnii/dashboard-react-tugas.git)

Langkah 2: Masuk ke Folder Proyek
Setelah berhasil di-unduh, masuk ke dalam folder proyeknya:

Bash
cd dashboard-react-tugas
Langkah 3: Install Semua Kebutuhan (Library)
PENTING: Kalian wajib menjalankan perintah ini supaya Tailwind CSS dan Framer Motion (animasinya) terpasang di laptop kalian. Kalau dilewat, web-nya bakal error atau tampilannya hancur!

Bash
npm install
(Tunggu sampai proses loading instalasinya selesai 100%)

Langkah 4: Nyalakan Server Web
Terakhir, jalankan perintah ini untuk menyalakan web-nya:

Bash
npm run dev
Setelah itu, buka browser (Chrome/Edge/Safari) dan ketik/klik tautan: http://localhost:5173/
